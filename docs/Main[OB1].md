# Main [OB1]

O bloco **Main [OB1]** é o bloco de organização cíclico principal do projeto no TIA Portal. Ele executa continuamente a lógica do programa a cada ciclo de varredura do CLP Siemens S7-1200. Neste projeto, o OB1 é responsável por realizar a chamada sequencial dos principais blocos de função (FBs) que compõem a lógica de comunicação, simulação e tratamento de alarmes. As chamadas foram implementadas em linguagem Ladder (LAD), garantindo clareza e padronização no fluxo de execução.
